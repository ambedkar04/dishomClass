# Tailwind CSS Theme for Django Unfold

This theme app integrates Tailwind CSS with Django Unfold admin interface.

## Setup Instructions

### 1. Install Node.js Dependencies

Navigate to the theme directory and install dependencies:

```bash
cd theme
npm install
```

### 2. Development Mode

To run Tailwind in watch mode during development:

```bash
npm run dev
```

Or using Django management command:

```bash
python manage.py tailwind start
```

### 3. Production Build

To build optimized CSS for production:

```bash
npm run prod
```

Or using Django management command:

```bash
python manage.py tailwind build
```

## File Structure

```
theme/
├── static/                  # Compiled CSS output
│   └── css/
│       └── styles.css      # Generated by Tailwind (gitignored)
├── static_src/             # Source files
│   └── src/
│       └── styles.css      # Tailwind source file
├── templates/              # Django templates
│   └── base.html
├── tailwind.config.js      # Tailwind configuration
├── postcss.config.js       # PostCSS configuration
├── package.json            # Node dependencies
└── .gitignore
```

## Customization

### Colors

Edit `tailwind.config.js` to customize colors that match your Unfold theme:

```javascript
theme: {
  extend: {
    colors: {
      primary: { ... },
      secondary: { ... }
    }
  }
}
```

### Custom Styles

Add custom styles in `static_src/src/styles.css` using Tailwind's `@layer` directive:

```css
@layer components {
  .my-custom-class {
    @apply bg-primary-600 text-white;
  }
}
```

## Integration with Django Unfold

The Tailwind configuration is set up to scan:
- All Django templates
- Python files (for Tailwind classes in forms, models, views, admin)
- JavaScript files

## Production Deployment

1. Build the CSS:
   ```bash
   npm run prod
   ```

2. Collect static files:
   ```bash
   python manage.py collectstatic --noinput
   ```

3. The compiled CSS will be served from `staticfiles/css/styles.css`

## Troubleshooting

### CSS not updating
- Make sure you're running `npm run dev` or `python manage.py tailwind start`
- Check that the file paths in `tailwind.config.js` are correct

### Classes not being purged in production
- Ensure your templates are included in the `content` array in `tailwind.config.js`
- Use complete class names (don't dynamically construct them)

### Django can't find the theme app
- Make sure `TAILWIND_APP_NAME = 'theme'` is set in `settings.py`
- Verify `'theme'` is in `INSTALLED_APPS`
